# 4D Pocket

4D Pocket 是一个为 Obsidian 设计的插件，用于创建和管理想法。

## 功能

- 快速创建带有时间戳的待办事项和想法
- 自动格式化想法条目
- 将光标定位到合适的位置以便快速输入

## 安装

1. 打开 Obsidian 的设置
2. 导航到 "第三方插件"
3. 确保 "安全模式" 已关闭
4. 点击 "浏览社区插件"
5. 搜索 "4D Pocket"
6. 点击 "安装"
7. 安装完成后，启用插件

## 使用方法

1. 在编辑模式下，使用命令面板（Ctrl/Cmd + P）
2. 搜索并选择 "创建Idea"
3. 插件会自动插入格式化的想法模板
4. 光标会自动定位到 "#idea" 标签之后，方便您立即输入想法内容

## 开发

如果您想为这个项目做出贡献：

1. 克隆这个仓库
2. 运行 `npm i` 安装依赖
3. 运行 `npm run dev` 以启动开发模式
